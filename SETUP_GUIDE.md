// ============================================
// STEP 1: INSTALL DEPENDENCIES
// ============================================
// Run in terminal:
// cd MyApp
// npm install

// ============================================
// STEP 2: SETUP FIREBASE
// ============================================
// 1. Go to https://console.firebase.google.com/
// 2. Click "Create Project"
// 3. Enter project name: "FastMadic"
// 4. Click through the setup
// 5. Create a web app (</> icon)
// 6. Copy the config object

// ============================================
// STEP 3: CREATE .env.local FILE
// ============================================
// Create file: MyApp/.env.local
// Add your Firebase config:

// EXPO_PUBLIC_FIREBASE_API_KEY=YOUR_KEY_HERE
// EXPO_PUBLIC_FIREBASE_AUTH_DOMAIN=YOUR_DOMAIN.firebaseapp.com
// EXPO_PUBLIC_FIREBASE_PROJECT_ID=your-project-id
// EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET=your-project.appspot.com
// EXPO_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=123456789
// EXPO_PUBLIC_FIREBASE_APP_ID=1:123456789:web:abc123

// ============================================
// STEP 4: SETUP FIRESTORE DATABASE
// ============================================
// In Firebase Console:
// 1. Go to Firestore Database
// 2. Click "Create Database"
// 3. Choose "Start in production mode"
// 4. Select region (e.g., us-central1)
// 5. Click "Create"
// 6. Replace security rules with the ones in README_COMPLETE.md

// ============================================
// STEP 5: SETUP STORAGE
// ============================================
// In Firebase Console:
// 1. Go to Storage
// 2. Click "Get Started"
// 3. Accept default settings
// 4. Update security rules for prescription image uploads

// ============================================
// STEP 6: CREATE TEST DATA (OPTIONAL)
// ============================================
// In Firestore Console, create these collections:

// Collection: shops
// Document ID: shop1
// {
//   name: "FastMadic Lahore",
//   region: "lahore",
//   contact: "03001234567",
//   adminId: "your_admin_uid"
// }

// Collection: users
// Document ID: (auto-generated by Firebase)
// {
//   name: "Admin User",
//   email: "admin@fastmadic.com",
//   phone: "03001234567",
//   role: "admin",
//   region: "lahore"
// }

// ============================================
// STEP 7: START DEVELOPMENT SERVER
// ============================================
// Run in terminal:
// npm run start

// Then press:
// - 'w' for Web (opens in browser)
// - 'a' for Android (requires Android Studio)
// - 'i' for iOS (requires macOS)
